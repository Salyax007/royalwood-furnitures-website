
# Royalwood Furnitures & Interiors

A responsive frontend website developed for **Royalwood Furnitures & Interiors**, a local furniture and interior design business.

## Project Overview

The client wanted a modern website that reflected the visual appeal and professionalism of leading furniture brands such as Stanley and Damro. The goal was to create a clean and attractive online presence that could showcase furniture collections, interior services, and business information.

The website was built using:

- HTML5
- CSS3
- JavaScript

## Features

- Responsive design
- Modern furniture-inspired UI
- Product showcase sections
- Service information
- Contact section
- Mobile-friendly layout

## Technologies Used

| Technology | Purpose |
|------------|----------|
| HTML5 | Structure |
| CSS3 | Styling & Layout |
| JavaScript | Interactivity |
